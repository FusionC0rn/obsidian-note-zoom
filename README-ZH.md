# Note Zoom

[English](README.md)

按住修饰键 + 滚轮，即可直观地缩放笔记内容。轻量、即时——每个标签页独立记忆缩放比例，侧栏不受影响。

*Mac 自动检测 ⌘ Cmd，Windows/Linux 默认 Ctrl。所有选项均可配置。*

---

## ✨ 功能

- **Ctrl/Cmd + 滚轮**缩放——轻量、即时的视觉缩放
- **双指捏合缩放**——精密触控板支持（需先点一下笔记区域激活）
- **按标签页记忆**——每个标签页独立记忆自己的缩放比例
- **按笔记记忆模式**（可选）——每个笔记文件独立记忆缩放，跨标签页切换不丢失
- **双击弹出滑块**——双击状态栏的 🔍 可拖出滑动条自由调节缩放
- **侧栏与 UI 不受影响**——仅笔记正文缩放，工具栏和面板保持清晰
- **状态栏指示器**——显示当前缩放百分比，单击循环预设，双击弹出滑块
- **3 条命令**——放大、缩小、重置缩放（可绑定快捷键）
- **5 种语言**——English、简体中文、繁體中文、日本語、한국어
- **完全可配**——修饰键、步长、范围限制、预设值、缩放模式、捏合灵敏度

## 📦 安装

### 手动安装

1. 从 [最新 Release](https://github.com/FusionC0rn/obsidian-note-zoom/releases) 下载 `main.js` 和 `manifest.json`
2. 在 `<vault>/.obsidian/plugins/note-zoom/` 创建文件夹
3. 将两个文件放入
4. 设置 → 第三方插件 → 启用 **Note Zoom**
5. 重载 Obsidian

### 从源码构建

```bash
cd <vault>/.obsidian/plugins
git clone https://github.com/FusionC0rn/obsidian-note-zoom.git note-zoom
cd note-zoom
npm install
npm run build
```

然后启用插件并重载。

## 🕹️ 使用

| 操作 | 方式 |
|------|------|
| 放大 | 按住修饰键 + 向上滚轮 |
| 缩小 | 按住修饰键 + 向下滚轮 |
| 弹出缩放滑块 | 双击状态栏的 🔍 |
| 循环预设 | 单击状态栏的 🔍 |
| 重置为 100% | `Ctrl+P` → `Reset zoom` |

三条命令均可在 **设置 → 快捷键** 中绑定自定义热键。

---

## ⚙️ 设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 界面语言 | 下拉 | English | 5 种语言：English / 简体中文 / 繁體中文 / 日本語 / 한국어 |
| 修饰键 | 下拉 | Ctrl (Win) / ⌘ Cmd (Mac) | 缩放时需按住的功能键 |
| 缩放步长 | 滑块 | 0.1 | 每次滚轮调整的幅度 |
| 捏合灵敏度 | 滑块 | 0.1 | 双指捏合的响应速度 |
| 最小缩放 | 滑块 | 0.3 | 允许缩小的最小比例 |
| 最大缩放 | 滑块 | 3.0 | 允许放大的最大比例 |
| 缩放模式 | 下拉 | 按标签页 | 标签页：每个标签页独立记忆。笔记：每个笔记文件独立记忆 |
| 预设缩放值 | 文本 | 0.8,0.9,1.0,1.1,1.2,1.5 | 逗号分隔；单击状态栏循环切换 |

---

## ⚠️ 已知限制

- **捏合缩放需先点击笔记区域**——Obsidian 的 Electron 运行时仅在内容区域获得焦点后才派发手势事件。捏合前先点一下笔记正文即可。
- **桌面端优先**——插件在移动端也可运行，但核心体验（修饰键 + 滚轮）需要物理键盘和鼠标/触控板。
